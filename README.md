#  Zellige Artisan – Landing Page

A modern, responsive landing page built for a Moroccan zellige artisan business.  
This project showcases traditional craftsmanship with a clean, premium web design.



## Overview

**Zellige Artisan** is a React-based landing page designed to represent a traditional Moroccan craft brand in a modern digital format.

The goal of this project is to combine:
- 🇲🇦 Moroccan cultural identity (zellige patterns & heritage)
- Modern UI/UX design principles
- Fast and responsive web performance



##  Features

-  Clean navigation bar with call-to-action
-  Hero section with decorative zellige-inspired background
- Animated-style statistics (experience, clients, projects)
-  Projects showcase grid
-  Call-to-action section for customer contact
-  Professional footer with contact details
-  Fully responsive layout (desktop-first, adaptable)


##  Tech Stack

- **React.js** – UI components
- **CSS3** – Custom styling (no UI frameworks)
- **JavaScript (ES6+)**
- **HTML5 semantics**
- **Supabase - Backend solution **

##  Design Concept

The design is inspired by traditional Moroccan zellige tiles:

- Geometric background patterns
- Deep teal & white contrast palette
- Warm orange accent color for CTAs
- Glassmorphism-style statistic cards
- Clean, modern typography



##  Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/zellige-artisan.git
````

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm start
```

The app will run on:

```
 http://localhost:5173/Zaliij_Craft/

```



##  Responsive Behavior

* Desktop: Full grid layout (4 project cards)
* Tablet: Adjusted grid (2 columns)
* Mobile: Stacked layout with centered content
